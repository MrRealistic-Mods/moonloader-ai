# 🌙 moonloader-ai

> AI reference files for writing bug-free **GTA San Andreas Moonloader** Lua mods.

---

## What's in here

| File | What it does |
|---|---|
| `CLAUDE.md` | Rules, patterns, and examples the AI must follow when writing scripts |
| `references.txt` | Every opcode and function from Moonloader, MoonAdditions, MoonPlus, ImGui, and Lua stdlib — ~4,000 entries grouped by category |

---

## What the reference covers

- **Moonloader** — 2,009 opcodes grouped by topic (Player, Ped, Vehicle, Camera, Weapons, HUD, Audio, World, etc.)
- **MoonAdditions** — 180+ extended functions including the full Vehicle Component API, Also includes opcodes from Dante's fork
- **MoonPlus** — 96 additional utility functions
- **ImGui** — 189 UI widgets for building in-game menus
- **Lua stdlib** — Standard functions available in the Moonloader context
- **Enums** — 1,325 named constants

---

## Requirements

- [Moonloader](http://gtaforums.com/topic/890987-moonloader/) installed in your GTA SA
- [MoonAdditions](https://github.com/Dante-1337/MoonAdditions) for extended functions (optional)
- [MoonPlus](https://github.com/THE-FYP/MoonPlus) for MoonPlus functions (optional)
